# Aspen Plus Simulation of Flash–Purge–Recycle System

## Overview

This project presents an Aspen Plus simulation for cyclohexane production via benzene hydrogenation.
The process includes flash separation, purge and recycle loops, and a distillation column for high-purity product recovery.

## Process Description

The flowsheet consists of:

* Feed preparation and heating
* Flash drum for vapor–liquid separation
* Purge stream to remove inerts
* Recycle loop to recover hydrocarbons
* Distillation column for final purification

## Key Results

* Achieved cyclohexane purity: **99.6–99.86%**
* Improved separation efficiency using recycle loop
* Reduced inert accumulation through purge stream

## Insights

* Recycle loop significantly enhances hydrocarbon recovery
* Purge stream prevents inert buildup and stabilizes system
* Distillation performance strongly depends on feed composition
